# Reverse Linked List
## **Problem Statment:**

Given the head of a singly linked list, reverse the list, and return the reversed list.

![img](https://assets.leetcode.com/uploads/2021/02/19/rev1ex1.jpg)

Example-1:  

```
Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]
```
